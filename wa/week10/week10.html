I found a cool API that simulates a card deck so I decided to make a relativly simple blackjack api.

The API returns a random card, so I made it when you click the "deal button" it grabs 4 cards, and store them either as dealer card or player card

Then when you hit it will pull another request and add it up with your current cards When you stand it will show the dealer card, and caluclate the winner
