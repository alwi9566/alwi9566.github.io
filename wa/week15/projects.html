Machine Learning Projects

Chess Performance Predictor & Style Analyzer

A machine learning system that analyzes my personal chess games from Chess.com to predict win probability and identify distinct playing styles using neural networks and clustering algorithms.

Project Overview

This project uses the Chess.com Public API to extract and analyze my game data, applying machine learning techniques to understand performance patterns and predict game outcomes. The system combines supervised learning (neural networks) with unsupervised learning (K-means clustering) to provide comprehensive insights into playing behavior.

Key Features

  • Win Probability Prediction: Neural network model predicting game outcomes based on rating differences, time controls, and playing patterns
  • Playing Style Classification: K-means clustering identifying distinct playing styles based on game characteristics
  • Opening Repertoire Analysis: Code analysis revealing most successful opening strategies
  • Temporal Performance Patterns: Analysis of performance variations by time of day and day of week
  • Feature Engineering: Extraction of 9+ features from raw game data including rating advantages, game length, and time controls

Technical Implementation

Data Collection & Processing:

  • Chess.com Public API integration for game retrieval
  • PGN (Portable Game Notation) parsing using python-chess library
  • Feature extraction from 6 months of games
  • Data cleaning and preprocessing with pandas

Machine Learning Models:

  • Neural Network Architecture: Sequential model with 64→32→16→1 neurons, ReLU activation, dropout regularization (0.3, 0.2), sigmoid output for binary classification
  • Model Performance: Achieved ~65% test accuracy with balanced precision and recall
  • Clustering Algorithm: K-means with 3 clusters, identifying aggressive, balanced, and defensive playing styles
  • Feature Scaling: StandardScaler normalization for optimal neural network performance

Key Technologies:

  • Python: Primary development language
  • TensorFlow/Keras: Deep learning framework for neural network implementation
  • scikit-learn: Machine learning utilities (clustering, preprocessing, evaluation)
  • pandas & NumPy: Data manipulation and numerical computing
  • python-chess: Chess game parsing and move extraction
  • matplotlib & seaborn: Data visualization and statistical plotting
  • Chess.com API: RESTful API for game data retrieval

Results & Insights

  • Most Predictive Features: Rating difference, color (white/black), and game length showed strongest correlation with win outcomes
  • Playing Styles Identified: Three distinct clusters representing different approaches to game selection and timing preferences
  • Opening Analysis: Identified best-performing openings with statistical significance (minimum 5 games played)
  • Temporal Patterns: Discovered performance variations based on time of day and weekend vs. weekday play

Future Enhancements

  • Interactive web dashboard for real-time win probability predictions
  • Expanded dataset with longer historical data
  • Deep learning model for opening-specific predictions
